# Inventory Management System

This repository contains the programming assignment for Advanced Programming Techniques.

## Repository Structure
To evaluate the impact of AI-assisted tools, this project has been developed in two distinct versions across different branches:

* Version A (Traditional Approach):** Located on the `main` branch. Developed manually without AI assistance.
* Version B (AI-Assisted Approach):** Located on the `version-b` branch. Developed with the assistance of AI tools-Claude Sonnet 5.



## Technical Specifications
* **Language:** Java
* **IDE:** IntelliJ IDEA
* **Data Storage:** CSV-based persistence (`inventory_data.csv`)
